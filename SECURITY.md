# Security Policy

It is unlikely this package would be a factor in any security-related scenario.  This package is nothing more than a feature-rich implementation of one ASN.1 construct: OIDs.

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| >=1.0.3 | ✅                 |
| < 1.0.2 | ❌                 |
